- "로그인" 링크 클릭 시: `/login` (S-01) 이동

## S-03. 할 일 목록 (`/`)

### 목적

인증된 사용자가 자신의 할 일 목록을 조회하고 상태/카테고리 필터를 적용한다. 목록 내 인라인 액션으로 완료 처리 및 삭제가 가능하다. (UC-05, UC-07, UC-08, BR-02, BR-08~BR-12)

### 데스크톱 레이아웃

```
+------------------------------------------------------------------------+
|  TodoList                                      [ 홍길동 ] [ 로그아웃 ] |  <NavBar>
+------------------------------------------------------------------------+
|                                                                        |

|  |                     |  |                                        |  |
|  |  상태               |  |  ← BR-09                              |  |
|  |  ( ) 전체           |  |  +------------------------------------+  |
|  |  ( ) 시작 전        |  |  | [시작 전] 리액트 공부하기          |  |
|  |  ( ) 진행 중        |  |  |  카테고리: 공부                    |  |  <TodoItem>
|  |  ( ) 완료           |  |  |  시작일: 2026-06-01                |  |
|  |  ( ) 기한 초과      |  |  |  종료일: 2026-06-30                |  |

|  |                     |  |  |  종료일: 2026-06-10                |  |

|  |                     |  |                                        |  |

|  |                     |  |  |  종료일: 2026-05-15                |  |
|  |                     |  |  |          [ 완료 ] [ 수정 ] [ 삭제 ] |  |  <Button x3>
|  |                     |  |  +------------------------------------+  |
|  |                     |  |                                        |  |
|  |                     |  |  ← BR-11                              |  |
|  |                     |  |  +------------------------------------+  |
```

### 상태 변형

- **로딩 상태**: 목록 영역에 스피너 또는 스켈레톤 카드 표시 (`<TodoList>` 내부)
- **빈 목록 (필터 결과 없음)**:
  ```
  +------------------------------------+
  |  조건에 맞는 할 일이 없습니다.     |
  |  [ + 새 할 일 등록하기 ]           |
  +------------------------------------+
  ```
- **빈 목록 (전체 없음)**: "아직 등록된 할 일이 없습니다. 새 할 일을 추가해보세요."
- **에러 상태**: "목록을 불러오지 못했습니다. 다시 시도해주세요." + `[ 새로고침 ]` 버튼

### 주요 인터랙션

- 상태 필터 선택: `GET /api/todos?status={status}` 호출 (BR-09~BR-12)
- 카테고리 필터 선택: `GET /api/todos?category_id={id}` 호출 (BR-08)
- "완료" 버튼 클릭: `PATCH /api/todos/:id/complete` 호출 → 상태 뱃지 [완료]로 변경, 완료 버튼 비표시 (UC-08, BR-05)
- "수정" 버튼 클릭: `/todos/:id/edit` (S-05) 이동
- "삭제" 버튼 클릭: 확인 다이얼로그 표시 후 `DELETE /api/todos/:id` 호출 → 목록에서 제거 (UC-07, BR-05)
- "+ 새 할 일" 버튼 클릭: `/todos/new` (S-04) 이동
- "홍길동" 클릭: `/profile` (S-06) 이동
